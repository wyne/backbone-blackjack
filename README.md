# Backbone BlackJack

## Running from github

Go to [http://wyne.github.io/backbone-blackjack/](http://wyne.github.io/backbone-blackjack/) and click Play Now.

## Running locally

Dependency: [NodeJS](http://nodejs.org/)

```
$ cd www
$ node ../server.js
```

Go to `http://localhost:8888`
